## Installation

Install the project with npm

```bash
  cd podcaster
  npm install 
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/JwRicauter/podcaster.git
```

Go to the project directory

```bash
  cd podcaster
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Deployment

To deploy this project run

```bash
  npm run deploy
```
